# Returns and Exchanges

Alpine Forge offers a 60-day return window on all unused, unworn items in original packaging. We want you to love what you buy — if something doesn't work out, send it back.

## How to return

1. Visit alpineforge.example/returns and enter your order number.
2. Select the items you want to return and the reason.
3. Print the prepaid UPS label we email you.
4. Drop the package at any UPS location within 14 days.

Refunds post to the original payment method within 5 business days of warehouse receipt.

## Exchanges

For size or color exchanges on apparel (Summit Shell Jacket, Ember Down Hoodie 800FP, Trailblazer Hiking Pants, Glacier Merino Base Layer, Torrent Rain Pants, Tundra Insulated Mittens, Zephyr Cycling Windbreaker), use the self-service exchange flow on alpineforge.example/exchanges. We'll ship the new size when you drop off the original at UPS — no waiting for it to reach the warehouse.

## Final sale items

Consumables and hygiene-sensitive items are final sale and cannot be returned. This is a short list:
- Open food or fuel
- Items marked "final sale" at checkout

## Used gear

We do not accept returns on clearly used gear outside the warranty process. If your item has a defect, see the Warranty policy.
